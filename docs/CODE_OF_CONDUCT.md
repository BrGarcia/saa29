# Código de Conduta

## Nosso Compromisso

Este projeto é desenvolvido em ambiente militar operacional. 
Esperamos de todos os colaboradores conduta profissional, técnica e respeitosa.

## Padrões Esperados

**Comportamentos que contribuem para um ambiente positivo:**
- Comunicação técnica objetiva e respeitosa
- Aceitação construtiva de críticas ao código
- Foco na qualidade do sistema e na missão operacional
- Documentação clara para facilitar a continuidade do projeto

**Comportamentos inaceitáveis:**
- Linguagem ofensiva ou desrespeitosa em qualquer meio (código, commits, PRs, comentários)
- Publicação de informações sensíveis ou classificadas em repositórios
- Sabotagem intencional de código ou documentação
- Personificação de outros membros da equipe

## Responsabilidades

Os responsáveis pelo projeto têm autoridade para remover contribuições que não sigam este código de conduta e tomar ações corretivas quando necessário.

## Aplicação

Casos de comportamento inadequado podem ser reportados diretamente ao responsável técnico do projeto. Todas as reclamações serão analisadas com confidencialidade.
