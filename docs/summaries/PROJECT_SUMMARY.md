# PROJECT_SUMMARY
Sistema de Acompanhamento de Aeronaves (SAA29).
Plataforma modular para controle de manutenção, panes e inventário da frota A-29.
Foco em rastreabilidade, auditoria (trigramas) e conformidade técnica.
Ambiente Dockerizado com Backend FastAPI (Python) e Frontend Jinja2/VanillaJS.
Status: Módulos de Panes, Vencimentos, Inventário e Inspeções totalmente integrados.
